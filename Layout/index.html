Ce este Crossfit

CrossFit este un program de pregătire fizică generală constituit din mișcări funcționale executate cu o intensitate ridicată ce variază constant. Adică un program care îți solicită toate cele zece abilități fizice generale: Anduranță Cardiovascular-Respiratorie, Acuratețe, Agilitate, Coordonare, Echilibru, Flexibilitate, Forță, Putere, Rezistență, Viteză. În prezent, CrossFit reprezintă cea mai complexă metodă de fitness.

CrossFit este un sport: sportul fitness-ului. Am învățat că valorificând camaraderia, competiția și amuzamentul ce rezultă din sport sau joc, obținem o intensitate ce nu poate fi atinsă prin alte mijloace.

Scopul unui antrenament de CrossFit este de a crea un program general și complex de fitness, program care prepară sportivul pentru orice eveniment fizic neprevăzut. Aruncând o privire de ansamblu la toate sporturile și la toate sarcinile fizice ale unui om, ne-am întrebat ce abilități fizice și de adaptare sunt necesare pentru a crea un avantaj competitiv și performanță. Informația culeasă din intersecția tuturor cerințelor sportive se aplică, în cel mai logic mod posibil, tuturor sporturilor. Cu alte cuvinte, specialitatea CrossFit este de a nu se specializa.

Este CrossFit pentru oricine?

Bineînțeles! Nevoile unui om obișnuit din punct de vedere sportiv și ale atleților olimpici diferă doar prin grad și nu prin fel. Cele zece abilități fizice generale sunt necesare atleților de calibru cât și oamenilor în vârstă. Este evident că nu vom încărca un vârstnic sau un om care nu a mai practicat sport cu aceeași greutate să facă genoflexiuni cum vom încărca un schior olimpic dar toți trebuie să le facă.

CrossFit este singurul program de fitness care a definit noțiunea de fitness într-un mod care poate fi măsurabil, observabil și repetabil.
